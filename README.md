# FocusFlow-Productivity-Tracker

## Overview of project:
FocusFlow is a productivity tool and workload tracking prototype designed to help college students manage assignments, track study sessions, and improve focus. 

This project is being developed as a capstone project for students at UMASS Boston


## Features in the productivity tracker:
- Focus session timer
- Weekly workload score calculator
- Distraction logging
- Productivity dashboard with analytics
- Assignment tracking system

## Technologies used:
- GitHub
- Netlify

## Target Market: 
This tool will be primarily for college students who want an easier way to handle their academic workload and lessen productivity loss due to distractions
